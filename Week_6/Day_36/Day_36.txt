# 1. Random Forest (RF)

# RF works by:

# averaging many diverse trees

# Main goal:

# variance reduction
# RF Core Ideas
# bootstrap sampling
# feature randomness
# decorrelated trees
# averaging predictions
# RF Deep Insight
# Diversity stabilizes prediction.

# Single tree:

# high variance
# unstable

# RF:

# stable
# generalized
# Important RF Hyperparameters
# Parameter	Role
# n_estimators	no. of trees
# max_depth	tree complexity
# max_features	randomness
# min_samples_leaf	overfitting control
# OOB Score

# OOB =

# internal validation using unseen bootstrap rows
# 2. XGBoost

# XGBoost works by:

# sequential error correction

# Each tree:

# corrects previous residuals
# XGBoost Core Ideas
# boosting
# residual fitting
# gradient optimization
# sequential learning
# Deep XGBoost Insight
# Boosting progressively refines decision boundaries.
# Important XGBoost Parameters
# Parameter	Role
# learning_rate	correction speed
# n_estimators	no. of refinements
# max_depth	complexity
# gamma	split restriction
# subsample	randomness
# reg_alpha	sparsity
# reg_lambda	stabilization
# Learning Rate

# Low LR:

# stable learning
# more trees needed

# High LR:

# aggressive optimization
# overfitting risk

# 4. Explainable AI (XAI)

# Feature importance:

# predictive usefulness

# NOT:

# causality
# SHAP

# SHAP explains:

# feature contribution to one prediction
# SHAP Formula
# Prediction = Baseline + Feature Contributions
# SHAP Values
# SHAP	Meaning
# Positive	increases prediction
# Negative	reduces prediction
# Global vs Local
# Global	Local
# overall behavior	individual prediction
# averaged logic	contextual logic
# Deep Explainability Insight
# Same fraud score can emerge through different behavioral pathways.
